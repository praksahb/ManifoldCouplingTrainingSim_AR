# Manifold AR Training System

A mobile AR module for training on manifold coupling systems, built with Unity and AR Foundation.

## Features

- **Login Screen**: Trainee entry and session tracking.
- **AR Placement Stage**: Plane detection; move, rotate, height adjust, confirm.
- **Connector Alignment**: UI feedback, nudge-to-align, event-driven progression.
- **Snap System**: Manual snap, orientation fix, state progression.
- **Handle Interaction**: Hold-rotate, lock/unlock detection, tutorial control.
- **Unsnap System**: Return to pre-snap, re-enable movement, emit events.
- **Tutorial Pipeline**: Data-driven steps, success/fail logic, hints, auto-advance.
- **Audio Cues**: Centralized, expandable event sounds.
- **Input Routing**: State-driven, clean separation between UI and gameplay.

## Not Implemented
- Directional UI Guidance Arrows: Logic present, visuals pending.
- Assessment Mode: Framework present (data/events), but lacks full flow and scoring.

## Installation

1. Clone the repo:
    ```
    git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY
    ```
2. Open in Unity 2020.3+ with AR Foundation.
3. Deploy to a supported device.

## Usage

- Enter info, follow AR placement and instructions, complete each tutorial step.
